electrode core

Electrode emphasizes convention overconfiguration, following industry best practices for performance, consistency, and reusability.

The heart of the platform is Electrode Archetypes. Archetypes allow for standardized configurations and structrue of the app and its associated react components. By enforcing a sensible structure and consistency for components, modules, and the entire app. Electrode's Archetype system ensures streamlined development and deployment.

electrode modules

Supercharge your app, with or without Electrode Core

  • Above-the-fold Rendering

    A react component that enables server side rendering of "only" the components your users see on the screen without scrolling. This way, instead of side rendering the whole page on the server, we only render what a user will see initially for improved performance.

  • Redux Router Engine

    Do you need data from a service to render your component server-side? This module fetches the data asyncronously and passes it to React to render your components. Requires Redux.

  • SSR Profiling & Caching

    Improve your React server-side rendering performance by up to 70% by caching your components' render output.

  • Confippet

    Lets you define, extend, and merge configurations for different environments (such as QA, prod, and staging) that get initialized during start up or at runtime.

  • Stateless CSRF Validation

    Validate HTTP requests without a session--for Express and Hapi.
